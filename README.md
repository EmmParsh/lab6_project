# Console File Manager

Консольный файловый менеджер — это  приложение на C++, которое позволяет работать с файлами и папками через текстовое меню в консоли.

## Возможности программы
- просмотр файлов и папок в текущей директории;
- отображение размера файлов;
- переход между папками;
- просмотр содержимого текстовых файлов;
- поиск файлов по названию;
- удаление файлов;
- копирование файлов.

## Используемые технологии
- C++;
- CMake;
- Git;
- GitHub;
- стандартная библиотека filesystem.
